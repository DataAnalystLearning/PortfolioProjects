Excel Dashboard

Welcome to my first Project Page

My very first project is creating an Interactive Dashboard in Excel. I generate IT Audits usually for schools. I use Excel because it is familiar with Head Teachers and Business Managers. Normally I would use a single worksheet with Conditional Formatting but I wanted to go a stage further and create an easy to use Dashboard.
